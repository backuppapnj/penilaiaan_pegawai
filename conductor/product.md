# Product - Vote System PA Penajam

## Product Name
Sistem Penilaian Pegawai Terbaik Pengadilan Agama Penajam

## Purpose
Sistem penilaian pegawai terbaik di lingkungan Pengadilan Agama Penajam dengan 3 kategori penilaian:
1. Pejabat Struktural/Fungsional
2. Non-Pejabat Struktural/Fungsional
3. Pegawai Disiplin (berdasarkan data absensi SIKEP)

## Target Users
- 29 Pegawai Pengadilan Agama Penajam
- 4 Role: Super Admin, Admin, Penilai, Peserta

## Key Features
1. Voting + Ranking system
2. 3 categories with different criteria
3. Excel import for discipline scores (SIKEP)
4. Digital certificate generation (DomPDF)
5. Multi-role dashboard
6. Anonymous voting with audit trail

- Deadline: Besok
