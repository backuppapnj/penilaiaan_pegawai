# Project Overview

**Project Name**: Vote System PA Penajam (Sistem Penilaian Pegawai Terbaik Pengadilan Agama Penajam)

## Purpose
A system to evaluate and vote for the best employees at Pengadilan Agama Penajam. It supports 3 categories:
1. Pejabat Struktural/Fungsional (Structural/Functional Officials)
2. Non-Pejabat Struktural/Fungsional (Non-Structural/Functional Officials)
3. Pegawai Disiplin (Discipline - based on attendance/SIKEP data)

## Target Audience
- 29 Employees of Pengadilan Agama Penajam.
- Roles: Super Admin, Admin, Penilai (Voter), Peserta (Candidate).

## Key Features
- **Voting & Ranking**: Weighted scoring system.
- **Data Import**: Excel import for discipline scores (SIKEP).
- **Certificates**: Automated PDF certificate generation for winners.
- **Dashboard**: Role-based dashboards.
- **Audit**: Anonymous voting with audit trails.
