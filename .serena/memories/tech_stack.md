# Tech Stack

## Backend
- **Language**: PHP 8.4+
- **Framework**: Laravel 12.x
- **Database**: MySQL
- **Auth**: Laravel Fortify v1
- **Testing**: Pest v4 (Unit, Feature, Browser)
- **Formatting**: Laravel Pint v1

## Frontend
- **Framework**: Inertia.js v2
- **Library**: React 19.x (TSX)
- **Styling**: Tailwind CSS v4
- **Routing/Type Safety**: Laravel Wayfinder v0
- **Linting/Formatting**: ESLint v9, Prettier v3

## Key Libraries
- **Excel**: `phpoffice/phpspreadsheet`
- **PDF**: `dompdf/dompdf`
- **QR Code**: `endroid/qr-code`
